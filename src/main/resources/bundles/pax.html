Pax URL Maven

Maven protocol enables you access to maven repository artifacts based on well known by now groupId, artifactId, version and type.

Notes

In the process of resolving the artifact the handler will make use of local repository if possible or any set of remote repositories including existing metadata for resolving latest of a snapshot version. To use the mvn url handler is not required to have maven installed on the machine that runs the osgi framework. Having maven installed will actually only have an impact on the download speed as if the referenced artifacts are already in the local repository, case when those artifacts will be used as opposite of going over the net and download them from a remote repository.

Install bundles from Maven repositories! mvn:groupId/artifactId/version
